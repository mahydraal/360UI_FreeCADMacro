# Code of conduct

## Our standard

Be respectful, technically specific, and patient with testers working across different operating systems, languages, FreeCAD builds, and experience levels.

Constructive behavior includes:

- focusing criticism on code and reproducible behavior
- distinguishing observations from assumptions
- sharing minimal test cases without confidential data
- acknowledging uncertainty and platform differences
- helping contributors understand FreeCAD concepts without ridicule

Harassment, personal attacks, discriminatory language, doxxing, and deliberate disruption are not acceptable.

## Enforcement

Repository maintainers may edit, hide, or remove content and restrict participation when necessary to protect contributors and project integrity. Report conduct problems privately through the security/contact route chosen by the repository owner.
