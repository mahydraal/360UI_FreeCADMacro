# Public-test roadmap

## Stabilization priorities

1. Confirm Sketch contextual ribbon behavior across Windows, macOS, and major Linux distributions.
2. Validate TechDraw model-view insertion and dimension selection on multiple FreeCAD 1.1 builds.
3. Collect Assembly solver diagnostics for representative joint graphs and linked-document workflows.
4. Stress-test modeled threads on common metric and Unified sizes.
5. Improve automated source-level validation without pretending to replace live FreeCAD tests.

## Candidate future work

- Preference dialog for shortcut and ribbon customization
- export/import of profile settings
- stronger document observers for immediate context changes instead of polling where FreeCAD APIs permit
- repair assistant for broken projection, dimension, thread, and joint references
- richer drawing sheet presets and standards
- structured diagnostic export for issue reports
- optional package-manager compatible add-on layout

Roadmap entries are proposals, not commitments.
