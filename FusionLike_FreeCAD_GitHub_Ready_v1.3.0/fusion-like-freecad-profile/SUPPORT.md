# Support

This is a community public-test project, not an Autodesk or official FreeCAD support channel.

Use GitHub Issues for reproducible defects and compatibility results. Use Discussions for workflow questions, design proposals, and help that does not yet have a minimal reproduction.

Include FreeCAD version, operating system, exact command sequence, selection context, and Report-view output. Maintainers cannot safely diagnose a geometry or solver problem from a screenshot alone.
