# Changelog

All notable changes are documented here. The project uses semantic-style version labels, but public-test builds may still contain compatibility defects.

## [1.3.0] - 2026-07-20

### Added

- contextual Sketch ribbon independent of active workbench name
- delayed Sketcher action registration detection and grouped-command fallbacks
- guided dimensionable TechDraw model-view insertion
- explicit separation of model views from raster Active View snapshots
- Smart Dimension selection diagnostics
- Assembly Add Selected, copy, paste, paste-in-place, and duplicate workflows
- live native joint-task diagnostic panel
- preflight connector analysis and post-mate decoded solver reports
- public smoke-test macro

### Retained

- native threaded Hole workflow
- independent modeled/cosmetic Thread feature
- persistent projection with direct links and SubShapeBinders
- Design, Assembly, Drawing, timeline, navigation, command palette, and reversible restoration

## [1.2.0] - 2026-07-20

- added Fusion-like threaded Hole dialog and independent Thread feature
- expanded Assembly and Drawing workspace ribbons
- added drawing thread callouts, BOM placement, and PDF export helpers

## [1.1.0] - 2026-07-20

- added linked face/edge/sketch projection
- added defining/reference/intersection modes
- added binder-mediated imported and cross-Body references
- added direct finish-Sketch-to-Pad/Pocket transition

## [1.0.0] - 2026-07-20

- initial reversible Fusion-like UI profile
- workspace selector, command palette, shortcuts, navigation mapping, and feature timeline
